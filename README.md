# 🤖 Portfólio de Automação com IA

> Projetos reais de automação com Inteligência Artificial desenvolvidos no setor jurídico.  
> Cada projeto documenta o problema, a lógica da solução e o impacto mensurável gerado.

---

## 📊 Impacto geral

| Métrica | Antes | Depois |
|---|---|---|
| Pessoas necessárias para pesquisa e fichas | 5 funcionários | 0 (100% automatizado) |
| Tempo para pesquisa DJEN + 3 fichas | 1 dia inteiro | ~5 horas |
| Tempo para produção de conteúdo (Instagram) | 1 semana | 1 dia |
| Horas humanas economizadas por semana | — | ~200h/mês estimadas |

---

## 📁 Projetos

### 01 — Pesquisa de processos rentáveis no DJEN
**[Ver case completo →](projetos/01-pesquisa-djen/README.md)**

Automação que pesquisa diariamente o Diário da Justiça Eletrônico Nacional (DJEN) em busca de processos com potencial de cessão de créditos, filtrando os casos rentáveis automaticamente.

`Jurídico` `Web Scraping` `NLP` `Triagem automatizada`

---

### 02 — Geração de ficha de cliente via EPROC
**[Ver case completo →](projetos/02-ficha-eproc/README.md)**

Automação que acessa o sistema EPROC, extrai todas as informações necessárias de um processo e gera automaticamente a ficha completa do possível cliente para cadastramento.

`Jurídico` `Extração de dados` `Automação de formulário`

---

### 03 — Geração de ficha de cliente via ESAJ
**[Ver case completo →](projetos/03-ficha-esaj/README.md)**

Mesma lógica da automação do EPROC, adaptada para o sistema ESAJ (Tribunal de Justiça de SP), com suas particularidades de estrutura e campos.

`Jurídico` `Extração de dados` `Automação de formulário`

---

### 04 — Geração de ficha de cliente via PJE
**[Ver case completo →](projetos/04-ficha-pje/README.md)**

Automação equivalente para o sistema PJE (Processo Judicial Eletrônico), integrando os três sistemas em um pipeline unificado de cadastramento de clientes.

`Jurídico` `Extração de dados` `Automação de formulário`

---

### 05 — Automação completa de redes sociais (Instagram)
**[Ver case completo →](projetos/05-social-media/README.md)**

Pipeline de IA que cria, revisa e publica conteúdo no Instagram de forma autônoma — desde a ideia do post até a publicação, incluindo legenda, hashtags e imagem.

`Social Media` `Geração de conteúdo` `Multimodal` `Publicação automatizada`

---

## 🛠 Stack de ferramentas

- **IA:** Claude (Anthropic) · GPT-4o
- **Automação:** n8n · Make · Zapier
- **Sistemas jurídicos:** DJEN · EPROC · ESAJ · PJE
- **Social:** Instagram Graph API
- **Outros:** Python · Playwright · Puppeteer
